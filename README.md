# angler

Angler is a custom video game RPG format designed for flexibility. It is primarily suitable for complex, long games but
can be used for simpler games or tabletop settings as well depending on the chosen level of complexity, of which there
are three.

> Note: the character generator is a work-in-progress and is not currently available.

It is also a character generator that allows random generation of all elements as well as manually editing them. This
is simply a fully static SPA web app.
